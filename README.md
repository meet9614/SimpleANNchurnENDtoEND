# 🚀 Customer Churn Prediction (End-to-End Deep Learning Project)

This project is part of my practice to build **end-to-end deep learning systems**, covering everything from data preprocessing to deployment.

---

## 📌 Project Overview

The goal of this project is to predict whether a customer will churn (leave the bank) using an **Artificial Neural Network (ANN)**.

The project demonstrates a complete ML pipeline:
- Data preprocessing
- Model training
- Model saving
- Deployment using Streamlit

---

## 🧠 Tech Stack

- Python
- TensorFlow / Keras (Deep Learning)
- Scikit-learn
- Pandas, NumPy
- Streamlit (UI & Deployment)

---

## ⚙️ Features

- User-friendly web interface
- Real-time churn prediction
- Preprocessing pipeline integrated with model
- Fully deployed ML application

## 🧪 How to Run Locally

### 1. Clone the repository

git clone https://github.com/meet9614/SimpleANNchurnENDtoEND.git
cd SimpleANNchurnENDtoEND

### 2. Install dependencies

pip install -r requirements.txt

### 3. Run the app

streamlit run app_py.py
